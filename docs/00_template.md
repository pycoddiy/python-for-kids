# 00 Template

![Screenshot](../screenshots/00_template.jpg)

Script: `00_template.py`

A minimal starting point for new examples. Shows a basic Arcade window and event hooks you can extend.

## Run

```bash
python 00_template.py
```

## Ideas to try

- Change window title and size
- Draw a shape or text in `on_draw()`
