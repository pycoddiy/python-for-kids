# 01 Basic Drawing

![Screenshot](../screenshots/01_basic_drawing.jpg)

Script: `01_basic_drawing.py`

Draw simple shapes and text with Arcade. Great for learning coordinates and colors.

## Run

```bash
python 01_basic_drawing.py
```

## Controls

- None; just watch the drawing

## Concepts

- Coordinate system (origin at bottom-left)
- Colors, line width, fill vs stroke

## Ideas to try

- Change colors and positions
- Add your name in big friendly text
