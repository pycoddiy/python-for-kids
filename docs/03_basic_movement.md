# 03 Basic Movement

![Screenshot](../screenshots/03_basic_movement.jpg)

Script: `03_basic_movement.py`

Continuous movement and speed. Teaches how to update a position each frame.

## Run

```bash
python 03_basic_movement.py
```

## Controls

- Usually arrow keys or WASD

## Concepts

- Delta time and frame updates
- Velocity vs position
